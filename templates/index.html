RHP Notifier

RHP Notifier is a Mac OS X application that shows a status icon with the number of games waiting for your move at online chess site Red Hot Pawn.

Download RHP Notifier __VERSION__
Universal Binary, requires OS X 10.5 (Leopard) or greater.

RHP Notifier is free to download and to use. The source code is available.

I have tested it on an Intel Mac on OS X 10.5 and 10.6. Please note although it is a Universal Binary I have not tested it on a PowerPC Mac so I cannot guarantee whether it will work on one! Please let me know.
